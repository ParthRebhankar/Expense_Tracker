# 💰 Student Expense Manager

A modern web application that helps students **track, analyze, and control their daily expenses** while studying away from home.

This project provides a simple yet powerful way to record transactions, monitor spending habits, and stay within budget.

Built using a **full-stack JavaScript architecture** with **React, Express.js, PostgreSQL, and ShadCN UI**.

---

## 🚀 Project Overview

Many students who move to a different city for studies struggle to manage their finances effectively. Small daily expenses quickly add up, and without tracking them properly, it becomes difficult to understand where the money goes.

This project solves that problem by providing a structured expense management platform where users can record their spending, organize expenses by category, and receive alerts when their spending crosses certain limits.

The goal of this project is to provide a **clean, easy-to-use expense management system designed specifically for students**.

---

## ✨ Features

### 📊 Expense Tracking
Users can add expenses with details such as:
- Category
- Amount
- Payment Method
- Notes

This allows them to maintain a clear record of their daily transactions.

### 🗂 Category Based Spending
Expenses can be categorized into different types such as:
- Food
- Travel
- Shopping
- Bills

This helps users understand where most of their money is being spent.

### 💳 Multiple Payment Methods
Users can record how the payment was made:
- Cash
- UPI
- Card

### 📈 Budget Monitoring
The application monitors spending and generates alerts when spending approaches or exceeds predefined limits.

### 🔔 Notification System
Users receive notifications for important financial events such as:
- Budget limit reached
- Budget exceeded
- High spending warnings
- Weekly expense review reminders

### 👤 User Authentication
Each user has their **own account and private data**, ensuring that expenses are stored securely and separately.

### 📜 Notification History
Users can view past notifications including:
- Budget alerts
- Expense warnings
- System messages
